# isotopes-core

![Python](https://img.shields.io/badge/python-3.11+-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Minimal, pluggable Python agent loop engine.

## Development

```bash
# Install uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Sync dependencies
uv sync --all-extras

# Run tests
uv run pytest

# Type check
uv run mypy src/

# Lint
uv run ruff check src/
```
